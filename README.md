# 🌌 Zodiac Sign Finder

A web application to discover your zodiac sign by birthday.

---

## ✨ Features

- 🔍 Search zodiac sign by birthday
- 🌟 View all 12 zodiac signs
- 📖 Zodiac detail pages
- 🎨 Beautiful Galaxy UI
- 📱 Responsive Design
- 🔗 REST API Integration

## 📡 API

- 📄 GET `/zodiacs`
- 🔍 GET `/zodiacs/search?name=Scorpio`
- 🎂 GET `/zodiacs/birthday?month=10&day=27`
